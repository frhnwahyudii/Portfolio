Academic Project

OOP Cashier Application

Cashier application developed as a logic implementation in Object Oriented Programming course using C# programming language. This project demonstrates basic OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction in desktop application development.

C# OOP Windows Forms .NET Framework Desktop Application

Key Features

Transaction Processing

Sales transaction input system with automatic calculations and data validation

Auto Calculation

Automatic calculation of total price, tax, discount, and change with high accuracy

User Interface

Intuitive and user-friendly desktop interface using Windows Forms

OOP Implementation

Complete implementation of OOP concepts: encapsulation, inheritance, and polymorphism

Product Management

Product data management with inventory system and stock tracking

Receipt Generation

Automatic receipt generation with neat format and complete information
